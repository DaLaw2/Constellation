# Constellation v1.1.2 Release Notes

Performance and usability update focused on faster thumbnail loading, batch tag operations, and improved search experience.

## ⚡ Performance

### 🖼️ Faster Thumbnail Loading
- **Parallel thumbnail generation**: Thumbnails now load significantly faster by utilizing multiple worker threads simultaneously.
- **Smoother scrolling in large directories**: Grid views now only render items visible on screen, eliminating lag when browsing folders with hundreds of files.
- **Configurable performance**: Fine-tune worker thread count and concurrency limit in Settings > Advanced to match your hardware.

## ✨ New Features

### 🏷️ Batch Tagging
- **Multi-select files**: Select multiple files with Ctrl+Click (toggle) and Shift+Click (range) in both Detail and Large Icon views.
- **Bulk tag operations**: Add or remove tags across all selected files at once.
- **Common tags display**: Instantly see which tags are shared across your selection.

### 🔍 Search Enhancements
- **Collapsible filter panel**: Collapse the search filter area to see more results.
- **Large Icons in search results**: Browse search results in Large Icon view.
- **Editable path bar**: Click the path bar to type a path directly for quick navigation.

### ⚙️ Advanced Settings
- New **Advanced** tab in Settings for thumbnail performance tuning:
  - Worker thread count (Auto / 1–16)
  - Concurrency limit (Auto / 2–32)
  - Force Windows Shell cache option (moved from Preview tab)

## 🧹 Improvements
- **Folder thumbnails**: Folders now display preview thumbnails.
- **Better file name display**: Grid view now shows up to 4 lines for long file names.
- **Smarter sidebar behavior**: Batch action bar hides when sidebar is expanded; clicking a search result auto-collapses the sidebar.

## 📦 Installation
Download the Windows installer and run it directly.
